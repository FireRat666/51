FireScreen Docs (WIP)

This documentation provides details for all parameters that can be used with the FireScreen script.
These parameters allow you to customize the behavior of the screen / browser, and UI elements.
If a parameter is not provided, its default value will be used.

Parameter Description Default Value Example Value
position Specifies the position of the screen in the space (x, y, z). 0 2 0 1 1 1
rotation Specifies the rotation of the screen in degrees (x, y, z). 0 0 0 45 90 0
disable-rotation Prevents the screen from rotating. Accepts true or false. false true
scale Determines the scale of the screen (x, y, z). 1 1 1 1.5 1.5 1
castmode Enables or disables cast mode. Accepts true or false
CastMode will disable most buttons but leave the top row.		 false true
lock-position Prevents the screen from being moved when set to true. false true
website Specifies the URL to load in the browser. https://firer.at/pages/games.html https://example.com
mipmaps Enable or disable mipmaps for screen textures. Accepts 1 (enabled) or 0 (disabled). 1 0
pixelsperunit Specifies the pixels per unit for rendering the screen. 1200 1400
width Width of the screen in pixels. 1024 1920
height Height of the screen in pixels. 576 1080
volumelevel Initial volume level (range: 0.0 to 1.0). 0.25 0.7
screen-position Relative position of the browser screen (x, y, z). 0 0 -0.02 0 1 -0.5
screen-rotation Rotation of the browser screen (x, y, z). 0 0 0 15 30 0
screen-scale Scale of the browser screen (x, y, z). 1 1 1 1.2 1.2 1
backdrop Enables or disables the screen's backdrop. Accepts true or false. true false
backdrop-color RGBA color of the backdrop. 0 0 0 0.9 1 1 1 0.5
button-color RGBA color for UI buttons. 0 1 0 1 1 0 0 1
volup-color RGBA color for the volume up button. 0 1 0 1 1 0.5 0 1
voldown-color RGBA color for the volume down button. 1 1 0 1 0 0.5 0.5 1
mute-color RGBA color for the mute button. 1 1 1 1 0.5 0.5 0.5 1
announce Enables general announcements. Accepts true or false. false true
announce-420 Enables 4:20 announcements. Accepts true or false. false true
announce-events Enables event-based announcements. Accepts true or false. false true
icon-mute-url URL for the mute button icon. https://firer.at/files/VolumeMute.png https://example.com/mute.png
icon-volup-url URL for the volume up button icon. https://firer.at/files/VolumeHigh.png https://example.com/volup.png
icon-voldown-url URL for the volume down button icon. https://firer.at/files/VolumeLow.png https://example.com/voldown.png
icon-direction-url URL for the back and forward navigation button icon. https://firer.at/files/Arrow.png https://example.com/arrow.png
custom-button01-url URL loaded when the first custom button is clicked. false https://example.com
custom-button01-text Text label for the first custom button. Custom Button 01 Home
custom-button02-url URL loaded when the second custom button is clicked. false https://example2.com/
custom-button02-text Text label for the second custom button. Custom Button 02 Feature 2
custom-button03-url URL loaded when the third custom button is clicked. false https://example3.com/
custom-button03-text Text label for the third custom button. Custom Button 03 Feature 3
custom-button04-url URL loaded when the fourth custom button is clicked. false https://example4.com/
custom-button04-text Text label for the fourth custom button. Custom Button 04 Feature 4
hand-controls Enable or disable hand controls for interacting with the screen. Accepts true or false. false true
disable-interaction Disables interaction with the screen if set to true. false true
space-sync Enable or disable synchronization with the space using Space State Properties. Accepts true or false.
Looks for the Protected Space State called 'fireurl' and will use it's value for the default page loaded		 false true

Example Usage

<script src="https://firer.at/scripts/firescreenv2.js"
  website="https://example.com"
  position="1 1 1"
  rotation="0 45 0"
  volumelevel="0.5"
  button-color="1 0 0 1"
  announce="true" ></script>
